Kolorlando

Definition: Kolorlando is a 3D game. A voxel world in which you play a voxel character.

Legend: done not done not used (done but not implemented)

Characteristics

  1. Movement, double jump and crouch.
  2. Fly mode
  3. Debug mode (see collisions)
  4. Hotbar for quick item selection
  5. Chat box
  6. NPCs (walker, chaser, talker)
  7. Minimap
  8. Animals
  9. Pets
  10. Mounts / Vehicles

Screen

  1. Resizing
  2. Desktop mode, mobile-portrait mode and mobile-landscape mode

Camera

  1. 3D Camera
  2. Skyrim Camera

    3. Zoomable camera from First Person mode to Third Person Mode

  3. WoW Camera
  4. Lego LoL Camera

User Interface (UI)

  1. Hotbar
  2. Minimap
  3. Chat
  4. Consola
  5. Joysticks
  6. Panel menu

Player

  1. Inventory
  2. Customizable characters
  3. Equipment/wearables
  4. Tools/weapons/goxels (gltf models scaled 0.1 and centered on X Y Z)
  5. HUD with health bar, selected voxel and game mode
  6. Step-up movement over one-block obstacles
  7. Left and right punch
  8. Shooting

Mobile mode

  1. Left joystick for movement and right joystick for camera
  2. Pinch to zoom and switch camera mode.

Voxel Edition

  1. 12 different voxels
  2. Voxel placement and removal
  3. Creative mode and Survival mode

Commands

  1. /debugmode

    2. Shows debug information like collisions or raycasts

  2. /flymode

    3. (enables flying)

  3. /spawn

    4. (respawns player)

Multiplayer

  1. Presence
  2. Broadcasting
  3. Authentication
  4. Chat

Settings

  1. dark / light theme
  2. fullscreen
  3. Background music
