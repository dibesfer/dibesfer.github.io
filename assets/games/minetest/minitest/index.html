Minitest

A Minetest game

What is minitest?

  • An easy game to make texture packs for
  • Without duplicate items e.g. sand, silver sand, desert sand or tree, acacia tree, aspen tree.

Minitest tries to simplify "Minetest the game", here's how:

  1. Delete mapgen v6 support.
  2. Delete all biomes but grasslands.
  3. Delete all trees but default tree.
