Fonish

Fonetic inglish and mor

Transcription ruls 
th → d (this → dis, that → dat)
exception: keep "th" in core semantic roots when useful (think → thinc, enough → inof)

ph → f
k / ck → c

-tion → keep as "tion" (pronounced softly, do not convert to shion)

VOWELS (5 CORE)
A E I O U
ah → a
eh → e
ee → i
oh → o
oo → u

RULES
silent e → drop
y → i (sound-based)
w → u (water → uater)

oriGinal = original (and pronounced like Garfield to come back to latin) or orighinal to represent it English sound

SYLLABLE RULES
silent → drop (make → meic)
y → i (sound-based)
w → u (water → uater)
double letters → simplify (running → runin)

v is kept for soft v french sound

spelling → sound first

GRAMMAR 

Easy past = Did + verb
Easy future = uil + verb

infinite verbs -ing → in
-ed can be used insted of irregular uaiked (woke), saied (said)...)
Adjective: +li (lovely -> lofli)
Comparative: +er (cheaper -> chiper)
PROFESSIONS 
+er → occupation / agent noun / person who... verb
butcher → batcher
worker → uorcer
teacher → ticher
driver → draiver
runner → raner
builder → bilder
Possesive: +s (Ghon's haus)

VERBIFICATION
-fy → -fai

simplify → simplifai
amplify → amplifai
modify → modifai
clarify → clarifai

-y nouns/adjectives → -i (phonetic soft ending)
sorcery → sorceri
energy → energi
memory → memori


goal: English → phoneme skeleton
keep meaning, remove noise and add regularizations.

Note: Fonish is a horse run between latin and modern english. It doesn't miss germanic K
