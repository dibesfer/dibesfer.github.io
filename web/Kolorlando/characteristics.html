Kolorlando

Definition: Kolorlando is a 3D game. A voxel world in which you play a voxel character.

Legend: done not done not used (done but not implemented)

Characteristics

  1. 3D Camera
  2. Desktop mode, mobile-portrait mode and mobile-landscape mode
  3. Zoomable camera from First Person mode to Third Person Mode
  4. Movement, double jump and crouch.
  5. Fly mode
  6. Debug mode (see collisions)
  7. Hotbar for quick item selection
  8. Chat box
  9. NPCs (walker, chaser, talker)
  10. Minimap
  11. Animals
  12. Pets
  13. Mounts / Vehicles

Player

  1. Inventory
  2. Customizable characters
  3. Equipment/wearables
  4. Tools/weapons
  5. HUD with health bar, selected voxel and game mode
  6. Step-up movement over one-block obstacles
  7. Left and right punch
  8. Shooting

Mobile mode

  1. Left joystick for movement and right joystick for camera
  2. Pinch to zoom and switch camera mode.

Voxel Edition

  1. 12 different voxels
  2. Voxel placement and removal
  3. Creative mode and Survival mode

Settings

  1. dark / light theme
  2. fullscreen
  3. Background music
