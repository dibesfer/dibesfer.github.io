The dibesGallery

Dev&Art Museum

What it is?

The dibesGallery is a private collection of artwork made by dibesfer and other artists.

It is accessible only with a password code

In order to get the code you must send an email to dibesfer@gmail.com using this format

Subject: dibesGallery register
    name: John Smith
    email: johnsmith@example.com

I manage registrations manually, store them in an email white list and send them password code on convenience.
